## Cómo completar una misión

1. Leer el `briefing.md` de la misión
2. Leer los `objetivos.md` y requisitos previos
3. Lanzar el entorno con `docker compose up` (si aplica)
4. Completar los checkpoints en orden
5. Capturar la **FLAG** final
6. Redactar un writeup y publicar en tu GitHub
7. Reportar el avance en el repo del operador
