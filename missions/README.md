## Cómo completar una misión

1. Lee el briefing completo antes de lanzar cualquier herramienta
2. Documenta todo en tu bitácora desde el primer comando
3. Completa los 3 checkpoints de cada fase
4. Captura la flag en el formato `0xVKRAD{...}`
5. Entrega la flag en el panel del operador
6. Escribe el writeup para ganar XP adicional
7. Publica evidencia en tu GitHub personal
